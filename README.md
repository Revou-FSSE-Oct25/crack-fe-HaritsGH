# Noartument

## Overview

A simple tournament management system with no image. Aimed to be lightweight. Made for small scale tournament for now.

## Features

- Simple account registration
- Simple tournament creation and management
- Simple tournament participation management
- Simple tournament bracket score reporting

## Getting Started

First, run the development server:

*Requires you to already installed npm.*

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

<u>**Make sure your backend is running.**</u>

## Tech Stack

- Next.js (Framework)
- Tailwind CSS (Styling)
- Axios library (HTTP Client)
- React-Hook-Form library (Forms)
